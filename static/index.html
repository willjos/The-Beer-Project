Documentation

Beer lovers can access information about their favourite beers using this website.

🍺 The Beer Project 🍺 currently has four endpoints:

  • '/' - the current website page
  • '/beers' - raw JSON data for all 5500+ beers
  • '/beers/{key}' - raw JSON data for an individual beer, accessed via its key number.
  • '/search' - this is a search endpoint that takes keyword query parameters and returns raw JSON. Users can do a case-insensitive search for beers with descriptions containing one or more (comma separated) keywords. For example: '/search?keywords=coffee,MaLtY,english.
  • The '/search' endpoint is utilised in the search bar functionality below.
